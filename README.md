# Docker Practice Web Application

## Project Overview

This is a simple static web application created for practicing Git, GitHub, and containerization concepts.
The project contains basic HTML pages and a CSS file that together form a small website.

This repository is mainly used for learning:

* Git and GitHub workflow
* Managing project structure
* Container deployment practice
* Static website hosting inside a container

---

## Project Structure

```
my-web-app
│
├── index.html
├── about.html
├── contact.html
│
├── css
│   └── style.css
│
└── README.md
```

### Description of Files

| File          | Description                          |
| ------------- | ------------------------------------ |
| index.html    | Main homepage of the web application |
| about.html    | About page describing the project    |
| contact.html  | Simple contact information page      |
| css/style.css | Stylesheet used for page design      |
| README.md     | Project documentation                |

---

## Features

* Simple static website
* Multiple HTML pages
* Basic navigation between pages
* External CSS styling
* Lightweight and easy to deploy
